Projects

SIEM home lab

A small SOC environment: Wazuh on VirtualBox, with a Windows 11 machine enrolled as a monitored agent. Using it I dug into a critical Node.js vulnerability (CVE-2025-55130), traced 489 defense-evasion events to MITRE ATT&CK T1562.001, and worked through the endpoint's failed CIS benchmark checks. Findings are written up in a SOC-style report in the repo.

Wazuh · VirtualBox · Windows 11

Network traffic analyzer

Command-line tool that reads PCAP files with Scapy and flags things worth a second look: port scans, legacy protocols, traffic to known-bad IP ranges, and credentials sent in cleartext over HTTP or FTP. Outputs a severity-ranked report, and ships with synthetic sample captures for testing.

Python · Scapy

Phishing email analyzer

Parses raw emails — headers and body — and scores them 0–100 for phishing risk based on spoofed sender domains, disguised links, urgency wording, and Reply-To mismatches, with a short readable summary of what it found.

Python

Log anomaly detector

Normalizes Windows Security events and Linux auth.log entries into a single timeline, then uses sliding windows and z-scores to surface brute-force bursts, logins at unusual hours, new account creation, and privilege escalation. Each finding gets a 0–100 risk score.

Python · pandas · numpy
